== Description ==
This plugin creates widget and shortcode with YouTube subscribing channel form or button.

== Installation ==

1. Upload or extract the `youtube subscriber` folder to your site's `/wp-content/plugins/` directory. You can also use the *Add new* option found in the *Plugins* menu in WordPress.
2. Enable the plugin from the *Plugins* menu in WordPress.
3. Now the YouTube Subscribe widget and shortcode are available.

= Usage =

1. Go to Appearance > Widgets.
2. Enable YouTube Subscriber widget.
3. Enter widget title
4. Enter text, which appeals to users, optional;
5. Enter your YouTube nickname or channel name (Id).
6. If you entered channel name (Id), adjust the appearance of the button.
7. Select layout, number of subscribers(view or hide).
8. Hit save.

YouTube Subscriber can be created using shortcode:
* Shortcode parameter - nickname ([youtube-subscriber nickname=])
or
* Shortcode parameters - channel ([youtube-subscriber channelname='' layout='' subscribers=''])
* Shortcode parameters - channel ([youtube-subscriber channelID='' layout='' subscribers=''])
1. Click button YouTube Subscriber in visual editor.
2. Enter your YouTube nickname or channel name (Id) in the pop-up.
3. If you entered channel name (Id), adjust the appearance of the button.
4. Select layout, number of subscribers(view or hide).
5. Hit Ok.
6. Click Publish or Update.
